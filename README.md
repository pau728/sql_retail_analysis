SQL Retail Sales Analysis



Overview

This project analyses retail sales data using PostgreSQL and Power BI.

The goal is to demonstrate end-to-end data skills: schema design, querying, aggregation, and business reporting.



Tools Used

\- PostgreSQL

\- DBeaver

\- Power BI Desktop

\- Git \& GitHub



Dataset

&nbsp;Retail data including:

\- Stores

\- Products

\- Sales transactions



SQL Workflow

1\. Create tables and relationships

2\. Insert reference and transaction data

3\. Perform analysis queries

4\. Build reporting views for BI



Power BI Analysis

&nbsp;The Power BI report connects directly to PostgreSQL reporting views and includes:

\- Revenue by Store

\- Revenue by Product

\- Daily Revenue Trend



Key Insights

\- Central Store generates the highest revenue

\- Laptops are the top revenue product

\- Revenue fluctuates significantly day to day
